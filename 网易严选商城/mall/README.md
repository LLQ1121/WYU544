# 商城 
```

项目是跟着b站up主*老陈打码*学习的,仅用于学习。
项目使用vue和vant ui开发，偏向移动端。
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### API接口文件

assets/config/api.js
